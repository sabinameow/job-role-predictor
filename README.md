# Skill-Based Job Role Predictor

A supervised machine learning web application that predicts a job role based on a given set of skills and experience level.

## Project Structure
job-role-predictor/
│
├── data/
│   ├── hh_jobs_dataset.csv        # Raw dataset scraped from hh.ru (570 job listings)
│   └── cleaned_dataset.csv        # Cleaned dataset generated by notebook 01
│
├── notebooks/
│   ├── 01_eda_cleaning.ipynb      # Exploratory data analysis and data cleaning
│   └── 02_model.ipynb             # Model training and evaluation
│
├── app.py                         # Streamlit web application
├── requirements.txt               # All required Python libraries
└── README.md                      # Project documentation


## How to Run

1. Clone the repository
2. Install dependencies
pip install -r requirements.txt
3. Run notebook `01_eda_cleaning.ipynb` to generate `cleaned_dataset.csv`
4. Run notebook `02_model.ipynb` to train and save the model
5. Launch the web app
streamlit run app.py

## Dataset

570 job listings from hh.ru covering 10 job roles: Data Scientist, Machine Learning Engineer, Data Analyst, Business Analyst, Backend Developer, Frontend Developer, DevOps Engineer, QA Engineer, UX Designer, Product Manager.

Features include 60 binary skill columns, experience level, and skills count.

## Algorithms Used

- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

Both models are evaluated using k-fold cross-validation and compared using accuracy, F1-score, and confusion matrix.

## Skills Demonstrated

- Feature engineering: encoding experience, cleaning binary skill features
- Classification metrics: accuracy, F1-score, confusion matrix
- Model evaluation: k-fold cross-validation, model comparison
