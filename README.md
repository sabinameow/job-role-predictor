# Skill-Based Job Role Predictor

A supervised machine learning web application that predicts a job role based on a given set of skills and experience level.

## Project Structure

job-role-predictor/
├── data/
│   ├── hh_jobs_dataset.csv        # Raw dataset scraped from hh.ru (570 job listings)
│   └── cleaned_dataset.csv        # Generated by notebook 01 (not included in repo)
├── notebooks/
│   ├── 01_eda_cleaning.ipynb      # Exploratory data analysis and data cleaning
│   └── 02_model.ipynb             # Model training and evaluation
├── models/                        # Generated after running notebook 02 (not included in repo)
├── app.py                         # Streamlit web application
├── requirements.txt               # All required Python libraries
└── README.md                      # Project documentation

## Setup Instructions

### 1. Clone the repository
git clone https://github.com/yourusername/job-role-predictor.git
cd job-role-predictor

### 2. Create and activate virtual environment
python -m venv venv

# Mac/Linux
source venv/bin/activate

# Windows
venv\Scripts\activate

### 3. Install dependencies
pip install -r requirements.txt

### 4. Run notebook 01
Open and run all cells in notebooks/01_eda_cleaning.ipynb
This will generate data/cleaned_dataset.csv

### 5. Run notebook 02
Open and run all cells in notebooks/02_model.ipynb
This will generate the trained model files in the models/ folder

### 6. Launch the web app
streamlit run app.py

The app will open automatically at http://localhost:8501

## Dataset
570 job listings from hh.ru covering 9 job roles after preprocessing:
Business Analyst, Backend Developer, Frontend Developer, DevOps Engineer,
QA Engineer, UX Designer, Product Manager, Data Analyst, Data Science / ML.

Features include 60 binary skill columns and experience level.

## Algorithms Used
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

Both models are compared using 5-fold cross-validation.
Final model: SVM with C=10, kernel=rbf, achieving 0.51 accuracy on the test set.

## Skills Demonstrated
- Feature engineering: encoding experience, cleaning binary skill features, merging minority classes
- Classification metrics: accuracy, F1-score, precision, recall, confusion matrix
- Model evaluation: 5-fold cross-validation, hyperparameter tuning with GridSearchCV, model comparison
